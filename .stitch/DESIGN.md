# Academy Realtors Design System

## Product Intent
Academy Realtors is a learning platform for realtors. The interface should feel calm, premium, and direct. It must help a person choose a course, understand what is open right now, and return to learning without friction. Avoid the feeling of an admin panel, spreadsheet, or product demo.

## Platform
- Web
- Desktop-first with strong mobile behavior
- Dark interface by default

## Visual Direction
- Atmosphere: confident, focused, premium, uncluttered
- Personality: editorial product design rather than startup dashboard
- Composition: large type, clean spacing, strong section rhythm, very few decorative elements
- Motion: subtle and meaningful only

## Color Roles
- Canvas: near-black blue surface
- Surface: dark slate panels with soft elevation
- Primary action: vivid cyan-blue accent
- Secondary action: dark bordered buttons with bright hover edges
- Text primary: high-contrast off-white
- Text secondary: cool muted gray-blue
- Borders: thin cool borders, never heavy grid lines

## Typography
- Headlines should feel decisive and readable, never crushed
- Overlines must be quiet support text, not dominant visual noise
- Avoid long uppercase labels in narrow columns
- Line height should prevent single-letter wraps and broken rhythm
- Public copy should sound human, concrete, and direct

## Layout Principles
- Prefer cards, grouped panels, and vertical stacks over table-like marketing layouts
- Do not place feature labels in a thin fixed column
- Keep major sections to one strong idea each
- Use whitespace to separate ideas instead of adding more borders
- FAQ should feel integrated into the page story, not bolted on as a support widget

## Copy Rules
- Speak to the person as "вы"
- Avoid product-manager language like "пользователь", "контур", "формат", "обучающий трек"
- Prefer practical outcomes:
  - "видно, что открыто"
  - "можно вернуться с того же места"
  - "курс разбит по шагам"
- Keep promises concrete and believable

## Components
- Buttons: bright primary fill, dark outline secondary, small radius
- Panels: dark, layered, restrained glow only on highlighted blocks
- Feature blocks: title above description, never side-by-side like a spreadsheet
- Course previews: should read like program choices, not registry rows

## Anti-Patterns
- No table-like marketing sections
- No generic SaaS filler copy
- No overloaded metadata on the first screen
- No stiff badge walls
- No light theme fallback for public marketing screens
